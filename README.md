# stAIn

https://removestain.netlify.app/

Remove clothing stains using AI. Upload a stain photo, enter the fabric type, and get a clear, step-by-step cleaning guide.

## How It Works

The process is simple:
1.  **Upload an Image:** Snap a photo of the stain and upload it.
2.  **Specify Fabric:** Add one or more fabric types and their percentages (e.g., 80% Cotton, 20% Polyester).
3.  **Add Context (Optional):** Provide details on how the stain occurred (e.g., "spilled coffee") for a more accurate analysis.
4.  **Get AI Instructions:** Click "Analyze Stain" to receive a tailored guide, including a summary, a list of materials needed, and clear, step-by-step removal instructions, all available in multiple languages.
